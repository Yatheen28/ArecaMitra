##Areca Mithra — Disease Detection and Surveillance Model
AI-powered arecanut disease classification model built using EfficientNet-B0, developed as part of the Areca Mithra early warning platform for arecanut farmers in Karnataka.

##About This Repository
This repository contains the initial implementation of the EfficientNet-B0 disease detection model and the app. Core geospatial components including DBSCAN clustering, convex hull computation, and outbreak zone boundary formation are not included in the public repository, as these modules are subject to an active research paper currently under preparation.
